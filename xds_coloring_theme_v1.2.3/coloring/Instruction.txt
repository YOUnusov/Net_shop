# Установка шаблона

 1. Содержимое папки upload скопировать в корневую директорию сайта
 2. Загрузить файл coloring.ocmod.xml через Еxtension Installer (установщик расширений)
 3. В разделе Modifications (модификации) нажать кнопку refresh (обновить)
 4. В разделе «Система -> Настройки» во вкладке «Магазин» выбрать шаблон «Coloring» и сохранить изменения.

**Шаблон готов к работе!**

# Предварительная настройка шаблона

### Рекомендуемые размеры изображений

 - для "продуктовых" модулей - 180х180
 - для списка товаров - 220х220
 - для дополнительных изображений товара - 69х69
 - для изображений связанных товаров 180х180

### Создание пользовательских меню в шапке сайта

 1. Перейти в раздел "Дополнения -> Модули"
 2. Активировать модуль "XDS Coloring Header Menu"
 3. Для главного меню создать новый модуль "XDS Coloring Header Menu" с именем "header-menu", а в нем создать необходимое количество пунктов меню.
 4. Для меню тех. поддержки создать новый модуль "XDS Coloring Header Menu" с именем "help-menu", а в нем создать необходимое количество пунктов меню.

### Создание карты в подвале

 1. Активировать модуль "HTML код"
 2. Создать новый модуль "HTML код" с именем "footer-map" 
 3. В поле "Код" вставить код сгенерированный с помощью сервиса от Яндекс "Конструктор карт" https://tech.yandex.ru/maps/tools/constructor/
 4. Сохранить изменения.

### Создание иконок платежных систем в подвале

 1. Перейти в раздел "Система -> Дизайн -> Баннеры"
 2. Создать новый баннер с именем "footer-pay-icons" 
 3. Создать необходимое количество иконок
 4. Сохранить изменения.

### Создание замыкающей карусели на главной странице  

 1. Перейти в раздел "Дополнения -> Модули"
 2. Активировать модуль "Карусель"
 3. Создать новый модуль "Карусель" с именем "home-bottom-carousel" 
 4. Создать необходимое количество пунктов
 5. Сохранить изменения.


# Отключение шаблона

 1. В разделе «Система -> Настройки» во вкладке «Магазин» выбрать любой другой шаблон
 2. В разделе Modifications деактивируйте модификацию "Coloring Theme Modification"
